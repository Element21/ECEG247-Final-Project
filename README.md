# Human Factors Platform

**Objective**: Design a microcontroller-based human factors testing platform using the MSP430G2553, a 1602 LCD, push buttons, and LEDs which is capable of conducting multiple experimental trials and recording data.

Functional Baseline for the Reaction Timer Your system must successfully execute a reaction time test:

1. Display instructions on the LCD through a usable, well-designed interface.
2. Wait for the user to press a button to begin the experiment.
3. Run through 10 trials: For each trial, after a random time delay, light an LED. Record the exact response time it takes the user to press a button.
4. Briefly pause to show the individual trial result on the LCD, then begin the next trial.
5. After 10 trials have been completed, calculate and display the average reaction time.
